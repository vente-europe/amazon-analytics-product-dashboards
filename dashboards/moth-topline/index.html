This dashboard is an empty skeleton copied from the Atopic Skin (dermo-products) topline build. Drop the X-Ray CSVs and run the build script to populate it.

  1. Drop Moth-UK.csv into data/x-ray/UK/ (and the same for DE, ES, FR, IT)
  2. Open _build_topline.py and update SEGMENTS to match the actual moth-control segments in your X-Ray (e.g. Pheromone trap, Sticky trap, Repellent)
  3. Update XRAY_LINKS with the Google Sheet URLs (top-right header buttons)
  4. Run py _build_topline.py from this folder to regenerate index.html

Once data is in, this placeholder will be replaced by the standard topline view: market-by-market revenue/units, segment × market heatmap, top brands per segment, and X-Ray edit buttons in the header.
